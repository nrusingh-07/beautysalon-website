#  Beauty Salon Website

A modern and responsive full-stack Beauty Salon Website designed for salons, spas, and beauty businesses. This project provides an elegant user interface to showcase beauty services, pricing, salon information, and customer interaction features.

🌐 **Live Demo:** https://beautysalon-website.onrender.com/  
📂 **GitHub Repository:** https://github.com/nrusingh-07/beautysalon-website

---

##  Features:

- Responsive and attractive UI
- Beauty salon landing page
- Service showcase section
- Pricing/packages section
- Contact section
- Smooth navigation
- Mobile-friendly design
- Full-stack project structure

---

## 🛠️ Technologies Used:

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
